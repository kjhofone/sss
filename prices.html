즉시 업데이트 설정
GitHub PAT를 입력하면 모든 참여자가 즉시 업데이트 버튼을 사용할 수 있습니다.
PAT는 Supabase DB에 저장됩니다. (RLS로 로그인한 사용자만 접근 가능)
발급: GitHub 프로필 → Settings → Developer settings → Personal access tokens → Tokens (classic) → workflow 권한 체크
자동 업데이트: 평일 16:10 장 마감 후 전종목 자동 갱신
즉시 업데이트: PAT 설정 후 버튼 클릭 → 스터디 탑픽 종목만 즉시 갱신 (약 30~60초)
